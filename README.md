# trabalho_SD_GP1

Você foi designado para desenvolver um sistema de monitoramento dos pneus dos carros durante o GP.
Trabalho da disciplina de Sistemas Distribuidos IFMGOB.
O trabalho visa simular um sistema realista distribuído utilizando conceitos como:
Coletar dados de forma distribuída, Armazenar os dados de forma segura e escalável, Expor os dados em tempo real, Aplicar estilos de comunicação diferentes, Orquestrar tudo com Docker/Docker Compose.
